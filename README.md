# custom-winpython
Custom WinPython with preinstalled modelx and lifelib

## Base WinPython distribution

* [Winpython64-3.8.2.0.exe](https://github.com/winpython/winpython/releases/tag/2.3.20200319)

## Removed packages

* GDAL
* imageio-ffmpeg
* imageio
* pyproj
* astropy
* astroML
* pygame
* torch
* torchvision

## Installed packages

* modelx v0.5.0
* lifelib v0.0.14
* spyder-modelx v0.3.1
